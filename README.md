# Project 2 Final Report

This repository contains the LaTeX source files, figures, references, and supporting draft materials for the final report of the swarm monocular SLAM robot project.

## Repository Structure

- `main.tex`: main LaTeX source of the report.
- `sample.bib`: bibliography file used by the report.
- `figures/`: figures used in the report, including robot images, architecture diagrams, and 2D map visualizations.
- `mycode/`: related implementation code referenced by the report.
- `ideas/`: Chinese draft notes, outline materials, and intermediate writing iterations used during the preparation of the report.

## Notes

- The `ideas/` directory mainly stores Chinese draft content and working notes rather than the final polished English text.
- The final report content is maintained in `main.tex`.
- Some materials in this repository are intermediate writing records used to support report iteration and revision.

## Build

The report can be compiled with a standard LaTeX workflow, for example:

```bash
latexmk -pdf main.tex
```

## Purpose

This repository is used to track the development, drafting, revision, and final assembly of the project report.
